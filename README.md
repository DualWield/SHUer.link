# SHUer.link

> Inspired by BYR.wiki

使用Nuxtjs构建的校园导航网站。

# 开发

## 安装依赖

`yarn install` 或者 `npm install`

## 测试服务器

`npm run dev`
启动测试服务器后可通过`localhost:3000`访问

## 静态化

`npm run generate`
